# 🎮 PRODIGY_WD_03 – Tic Tac Toe Web Application

A modern and interactive Tic Tac Toe Web Application developed as part of the Prodigy InfoTech Web Development Internship. The project provides an engaging user experience with responsive design, smooth gameplay, and real-time winner detection.

## 🚀 Live Demo

🌐 Website:
https://prodigy-wd-03-six-wine.vercel.app/

## 📌 Project Overview

This project is a fully functional Tic Tac Toe game where players can compete against each other on a responsive and interactive game board.

The application includes game state management, winner detection logic, draw detection, and restart functionality while maintaining a clean and modern user interface.

## ✨ Features

- Interactive Tic Tac Toe Board
- Two Player Gameplay
- Real-Time Winner Detection
- Draw Match Detection
- Game Reset Functionality
- Responsive Design
- Smooth User Experience
- Mobile Friendly Interface
- Modern UI Design

## 🎮 Game Rules

1. The game is played on a 3×3 grid.
2. Player X starts first.
3. Players take turns placing their marks.
4. The first player to align three marks horizontally, vertically, or diagonally wins.
5. If all cells are filled without a winner, the game ends in a draw.
